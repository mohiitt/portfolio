TinyScout
January 2026 Built
Claude API Python Streamlit Multi-Agent Agentic RAG

An autonomous web research agent that turns a single goal into a structured, cited report. A Planner decomposes the objective into sub-tasks, a RetrieverFactory fetches from multiple backends with topic-aware cache invalidation, and an Analyzer/Synthesizer pipeline produces the final output — with an evidence gating layer that returns "Insufficient Evidence" signals instead of hallucinating answers when sources are weak.

View on Devpost ↗
Key Design Decisions
Evidence gating — structured signals over fabricated answers, with per-source credibility scoring and full retrieval traces surfaced in the dashboard.
Three-tier fallback — planner output parsed via JSON → list → single-task, with multi-backend fetch resilience to handle real-world 403/404 failures gracefully.
Thin-content filtering — retrieval results below a relevance threshold are discarded before reaching the synthesis stage.
